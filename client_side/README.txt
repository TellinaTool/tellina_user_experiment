Welcome to the Bash User Experiment!

In this experiment, you will be presented with a number of Bash specific file
system tasks to solve. The tasks are non-trivial in terms of difficulty and is
chosen such that you would have to use numerous tools (man pages, Google
searches, etc.) to solve them.

Make sure you have read the all the details on the website about the experiment
before beginning. Once you are ready, run
```sh
source ./configure
```
