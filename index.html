Welcome to the Bash User Experiment website!

For this experiment, you will be presented with a series of file system tasks that can be done within a Bash shell. For each task, you are encouraged to use a variety of tools (man pages, Google, etc.) to help you solve it.

For the experiment to go as smoothly as possible, please ensure that you are using a machine with a graphical display, Bash 3.0 or higher, and Meld installed.

Once all of those prerequisites are met, please download the following ZIP file: bash_user_experiment.zip
