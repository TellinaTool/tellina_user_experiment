Welcome to the English to Bash Experiment website!

For the experiment to go as smoothly as possible, please ensure that you have access to a CSE Linux Home VM with Meld installed. You can verify if Meld is installed by using the Meld command in your VM's terminal: $ meld. It should open a GUI window. To install Meld on your CSE Linux Home VM use $ sudo yum install meld.

You will be presented with a series of file system tasks that can be done within a Bash shell. For each task, you are encouraged to use a variety of tools (man pages, Google, etc.) to help you solve it. Your objective is to solve each task with a Bash one-liner: a bash script that is contained on one line. E.g., $ ls | tee file

Once all of those prerequisites are met and you are ready, please follow these step to start the experiment:

  1. Download experiment.zip in a convenient folder on your CSE Linux Home VM.
  2. Unzip with $ unzip experiment.zip
  3. Start the experiment $ source bash_experiment/configure
When you start the experiment, follow the prompts in your terminal. You will have 2 training questions to familiarize yourself with the tool and then the real tasks.

After the experiment
